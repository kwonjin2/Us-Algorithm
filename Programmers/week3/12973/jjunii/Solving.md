# 📝 Solving: [12973 짝지어 제거하기](https://school.programmers.co.kr/learn/courses/30/lessons/12973)

## 문제 요약

- `s`: 알파벳 소문자로 이루어진 문자열
- 문자열에서 같은 알파벳이 2개 붙어있는 짝을 찾아 제거 -> 앞뒤로 문자열 이어 붙임
- 위 과정을 반복해서 모든 문자열 제거 가능하면 1, 아니면 0 return
- `len(s)` <= 10^6

---

## 풀이 방법

- 이어진 짝을 찾아야 하므로 stack 사용
- 문자열을 차례대로 순회하며,
  1. `not stack`: 현재 가리키는 문자가 가장 첫 번째 문자 -> 해당 문자 stack에 append
  2. `stack[-1] == 현재 가리키는 문자`: 직전에 들어간 문자와 현재 가리키는 문자가 동일 즉, 이어진 문자가 동일 => `stack.pop()`으로 문자쌍 제거
  3. `stack[-1] != 현재 가리키는 문자`; 직전에 들어간 문자와 현재 가리키는 문자가 다름 즉, 이어진 문자가 다름 => 해당 문자 stack에 append
- 순회가 끝난 후,
  - stack이 비어있지 않다 == 쌍으로 제거되지 못한 문자가 남아있다 => 0 return
  - stack이 비어있다 == 모든 문자가 `pop()`으로 제거됨 => 1 return
