# [42577] 전화번호 목록

```python
def solution(phone_book):
    answer = True
    dict = {}
    for phone in phone_book:
        dict[phone] = True
        
    for phone in phone_book:
        for i in range(1, len(phone), 1):
            prefix = phone[0:i]
            if prefix in dict:
                return False
    
    return answer
```

## 접근방식
- 전화부에 적힌 전화번호 중 한 번호가 다른 번호의 접두어인 경우가 있는지 확인
- 구조대 : 119
- 박준영 : 97 674 223
- 지영석 : 11 9552 4421
- 먼저 딕셔너리에 True 값으로 저장한다.
- 반복문으로 하나식 슬라이싱해서 접두어 찾아 값 반환
