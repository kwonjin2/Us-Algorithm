```python
def solution(n):
    MOD = 1000000007
    if n == 1: return 1
    if n == 2: return 2

    a, b = 1, 2  # dp[1], dp[2]
    for _ in range(3, n+1):
        a, b = b, (a + b) 
    return b  % MOD
```

## DP 접근 방식 두 가지
1. 메모이제이션 (Top-Down)
- 재귀 호출 + 캐싱
- 큰 문제를 풀면서 필요한 작은 문제를 재귀적으로 호출

2. 타뷸레이션 (Bottom-Up)
- 작은 문제부터 차례대로 테이블(배열)에 값을 채워 나감
- 반복문으로 구현되는 경우가 많음
- 재귀 호출 X, 스택 오버플로우 걱정 없음
