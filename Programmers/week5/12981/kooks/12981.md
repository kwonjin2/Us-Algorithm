# [12980] 점프와 순간 이동

```java
// k칸 점프해서 현재까지 온거리 * 2 위치로 순가 이동 
// k 칸을 점프하면 K만큼의 건전지 사용량이 든다.
// 순간이동은 건전지 사용량이 줄어 들지 않는다.
// n 거리가 주어졌을 때 사용량 최솟값을 리턴하는 함수
// 0 -> 5 : 5
// 0-2 

import java.util.*;

public class Solution {
    public int solution(int n) {
        int ans = 0;

        while(n > 0){
            if(n % 2 == 0){
                n /= 2;
            } else {
                n -= 1;
                ans++;
            }
        }

        return ans;
    }
}
```
