# [70129] 이진 변환 반복기

```java
// 문자 s의 0을 모두 제거 후 제거 횟수를 이진 변환한다.
// 다시 0을 제거... 반복
// 마지막 1이 될때 까지의 반복 횟수

class Solution {
    public int[] solution(String s) {
        int[] answer = {0, 0};
        
        String binary = s;
        
        while(!binary.equals("1")){
            int count = 0;
            
            for(char ch : binary.toCharArray()){
                if(ch == '0'){
                    answer[1] += 1;
                }else{
                    count += 1;
                }
            }
            answer[0] += 1;
            binary = Integer.toBinaryString(count);
        }
        
        return answer;
    }
}
```
