# 📝 Solving: 의상 (42578)

## 문제 요약
- 코니는 각 종류별로 최대 1가지 의상만 착용할 수 있다. 예를 들어, 얼굴 종류에서 동그란 안경, 검정 선글라스를 동시에 착용할 수 없다. 코니는 하루에 최소 1가지 의상은 입는다.
이때, 코니가 가진 의상들이 담긴 2차원 배열 clothes가 주어질 때 서로 다른 옷의 조합의 수를 return 하도록 solution 함수를 작성해라

## 접근 방법
- 먼저 문제를 정확히 이해하려면 예시를 살펴봐야 한다.
    ```js
    [
        ["yellow_hat", "headgear"],
        ["blue_sunglasses", "eyewear"],
        ["green_turban", "headgear"]
    ]
    ```
    - 이러한 데이터가 들어온다고 가정하면, 코니는 다음과 같이 경우의 수를 구할 수 있다.
        - yellow_hat
        - blue_sunglasses
        - green_turban
        - yellow_hat + blue_sunglasses
        - green_turban + blue_sunglasses
    - 즉, 5가지의 경우의 수이다.

- 각 데이터는 옷, 옷의 타입으로 들어오게 된다. 이때, 옷 타입이 몇개인지 알면 총 경우의 수를 구할 수 있다.
    - 자세히 말하면 위의 예시에서는 headgear:2, eyewear:1이다. 
    - 이때 코니가 입는 총 경우의 수는 5이다.
        - 총 경우의 수 : (headgear + 안 입는 경우(1)) * (eyewear + 안 입는 경우(1)) = 6 - 1(코니는 의상 종류 중 아무것도 걸치지 않으면 안된다는 조건);

- 이제 총 경우의 수를 구하는 방법도 알았으니, 구현만 하면 된다. 코니의 옷 종류와 개수를 세기위해 map을 이용했다.

- map을 통해 해당 type(옷의 종류)에 접근하여, 이 값이 없다면 0으로 초기화한 뒤에 1을 더해줬다. 즉, 옷의 종류에 해당하는 옷의 개수를 셀 수 있다.

- 이렇게 옷 종류의 각 개수를 세줬다면, 이 값을 통해 코니가 입을 수 있는 경우의 수를 구해줘야 한다. 위 설명에서와 같이 모든 경우의 수는 옷 종류 개수 + 1(안 입는 경우의 수)를 각 종류별로 곱해서 모든 경우의 수를 구한다. 그리고 코니가 아무것도 안 입는다면 그대로 return 해줘도 무방하지만, 문제에서 코니가 하루에 최소 1가지의 의상은 입는다고 명시되어 있으니, 구한 경우의 수에서 -1한 값을 return한다.
