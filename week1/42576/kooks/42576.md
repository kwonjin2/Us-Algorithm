# [42576] 완주하지 못한 선수

## 1차 시도
```python
def solution(participant, completion):
    answer = ''
    
    for completion_name in completion:
        for participant_name in participant:
            if participant_name == completion_name:
                participant.remove(participant_name)
    
    return participant[0]

# 마라톤 완주를 하지 못한 1명의 선수를 리턴
# 참가 a,b,c,d
# 완주 a,b,c
# 결과 1,1,1,0
# a 선수가 완주 했나? ture
# b 선수가 완주 했나? ture
# c 선수가 완주 했나? ture


```

- 효울성에서 통과하지 못함
- O(N^2) 시간 복잡도를 가진다.

## 2차 시도
```python
def solution(participant, completion):
    participant.sort()
    completion.sort()
    
    for p, c in zip(participant, completion):
        if p != c:
            return p
    
    return participant[-1] 
```
